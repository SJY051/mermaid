# 발표 노트 — 만든 사람의 문장으로

태스크를 끝낼 때마다 여기에 **3줄 + 시연 장면 1개**를 적습니다. 형식은 아래 예시처럼, 자기 말로.
이 파일이 3주 뒤 발표 대본의 원료가 됩니다 — 마지막 주에 몰아 쓰는 대본이 아니라, 만든 순간의 문장으로.

관리: 최정민 (PM/QA) · 작성: 만든 사람 본인 · 규칙: [tasks.md §4.5](specs/001-foundation/tasks.md)

---

## 예시 (윤서진 · 2-패스 RAG · DEV-403)

- AI가 아는 약을 말하는 게 아니라, AI가 "찾아볼 성분"만 고르고 실제 정보는 식약처 API에서 가져오게 만들었다.
- 모델이 조회되지 않은 약 이름을 지어내면 서버가 답변을 통째로 거부한다 — 환각이 사용자에게 닿을 수 없다.
- "I have a headache"라고 치면 실존하는 제품 3개가 정부 데이터 출처와 함께 나온다.

**시연 장면:** 증상 입력 → 실제 약 카드 3장, 각 카드에 식약처 출처 표시가 뜨는 순간.
