# 시작하기 — 사람을 위한 안내

이 문서는 **사람**용입니다. 규칙 전문과 AI 어시스턴트 지침은 [AGENTS.md](../AGENTS.md)(영어), 바로 실행해 보는 법은 [README](../README.md)에 있습니다. 여기서는 **무엇이 궁금할 때 어디를 보면 되는지**, 그리고 **각자 오늘 무엇부터 하면 되는지**만 다룹니다.

## 우리가 만드는 것 (30초)

한국에 있는 영어 사용자가 로그인 없이 영어로 증상을 말하면 → **정부 공공데이터로 검증된** 약 정보를 영어로 설명하고, 지금 열려 있는 근처 약국·병원을 지도로 보여줍니다.

AI가 하는 일은 딱 두 가지입니다: *어떤 성분을 찾아볼지 정하기*, *찾아온 사실을 영어로 풀어쓰기*. 사용자에게 보여주는 사실은 전부 식약처·심평원·국립중앙의료원 API에서 옵니다. AI가 목록에 없는 약 이름을 지어내면 서버가 그 답을 통째로 버립니다.

코드를 안 짜는 사람도 GitHub 웹에서 바로 기여할 수 있습니다: 파일 열기 → 연필 아이콘 → 고치기 → "Create a new branch for this commit" 선택 → PR. 터미널이 필요 없습니다.

## 세 가지만 기억하세요

1. **`main`에 직접 푸시하지 않기.** 브랜치(`feat/DEV-203-...`) 만들고 PR로.
2. **"완료"는 테스트가 지금 통과하는 것.** `cd backend && ./gradlew test`, `cd frontend && pnpm test` — 기억이 아니라 방금의 초록불.
3. **30분 막히면 물어보기.** 질문이 이 팀에서 가장 싼 행동입니다.
