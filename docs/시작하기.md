# 시작하기 — 사람을 위한 안내

이 문서는 **사람**용입니다. 규칙 전문과 AI 어시스턴트 지침은 [AGENTS.md](../AGENTS.md)(영어), 바로 실행해 보는 법은 [README](../README.md)에 있습니다. 여기서는 **무엇이 궁금할 때 어디를 보면 되는지**, 그리고 **각자 오늘 무엇부터 하면 되는지**만 다룹니다.

## 우리가 만드는 것 (30초)

한국에 있는 영어 사용자가 로그인 없이 영어로 증상을 말하면 → **정부 공공데이터로 검증된** 약 정보를 영어로 설명하고, 지금 열려 있는 근처 약국·병원을 지도로 보여줍니다.

AI가 하는 일은 딱 두 가지입니다: *어떤 성분을 찾아볼지 정하기*, *찾아온 사실을 영어로 풀어쓰기*. 사용자에게 보여주는 사실은 전부 식약처·심평원·국립중앙의료원 API에서 옵니다. AI가 목록에 없는 약 이름을 지어내면 서버가 그 답을 통째로 버립니다.

| 최정민 | PM/UX/QA | **DEV-305 동의어 사전 검토** — [검토 시트](specs/001-foundation/DEV-305-synonym-review.md)에 판정만 하면 되게 준비돼 있습니다. 이 칸은 사람만 채울 수 있어요 |
| 윤서진 | BE-1·FE-1 / 리드 | 챗·RAG·CI. 막히면 이 사람에게 물어보세요 |

코드를 안 짜는 사람도 GitHub 웹에서 바로 기여할 수 있습니다: 파일 열기 → 연필 아이콘 → 고치기 → "Create a new branch for this commit" 선택 → PR. 터미널이 필요 없습니다. 자세한 순서와 **봇 리뷰가 떴을 때 할 일**은 [PR 가이드](PR-가이드.md)에 있습니다.

> PR을 올리면 **Codex 봇이 자동으로 리뷰**합니다. 사소한 스타일은 짚지 않게 설정돼 있어서, 봇 코멘트가 떴다면 진짜 봐야 하는 것입니다. 무슨 말인지 모르겠으면 물어보세요 — 그것도 정상적인 질문입니다.

## 코딩 에이전트를 쓴다면

쓰세요 — 다들 씁니다. 다만 우리 팀에서 실제로 자주 나는 사고 세 가지만 피해 주세요. **① 이해 없이 대량 생산**: 에이전트가 쓴 코드를 자기 말로 설명 못 하면 PR을 올리지 마세요(첫 과제로 에이전트에게 "이 레포와 내 레인을 설명해줘"라고 시키는 게 최고의 시작입니다). **② 빨간 테스트를 테스트 수정으로 끄기**: 테스트가 실패하면 코드를 의심하는 게 먼저입니다 — 에이전트가 "테스트를 고치겠다"고 하면 멈추고 리드와 상의하세요. **③ 에러 메시지 붙여넣고 나온 걸 무검토 적용**: 에이전트의 수정도 남의 PR 보듯 읽고 적용하세요. 자세한 규칙은 [AGENTS.md §13-6](../AGENTS.md#13-6-three-rules-when-the-agent-writes-the-code)에 있습니다.

## Windows 쓰신다면

다 됩니다. 규칙 하나만: **저장소의 `.sh` 스크립트는 Git Bash에서 실행하세요.** Git for Windows를 깔았다면 이미 들어 있습니다. 나머지(`pnpm`, `docker compose`, `gradlew.bat`)는 PowerShell에서 그냥 돌아갑니다.

`bin/check-api-access.py`는 `python bin/check-api-access.py`로 부르세요.

줄바꿈(CRLF)과 심링크 함정은 저장소 쪽에서 미리 막아뒀습니다([AGENTS.md §3](../AGENTS.md#on-windows)). 다만 **아직 실제 Windows에서 돌려본 적이 없어요.** 처음 클론하시는 분이 `./bin/setup.sh` → `./gradlew test` → `pnpm test`까지 해보고, 깨진 게 있으면 알려주세요. 다음 사람이 안 겪습니다.

## 세 가지만 기억하세요

1. **`main`에 직접 푸시하지 않기.** 브랜치(`feat/DEV-203-...`) 만들고 PR로.
2. **"완료"는 테스트가 지금 통과하는 것.** `cd backend && ./gradlew test`, `cd frontend && pnpm test` — 기억이 아니라 방금의 초록불.
3. **30분 막히면 물어보기.** 질문이 이 팀에서 가장 싼 행동입니다.
