## Attack Path Steps

1. Attacker or external source controls the validated input: Prompt- or repository-influenced Git command text controls global options before a diff/show operation on .env.
2. The stated preconditions hold: A populated readable .env exists, the hook is active, and the agent executes the crafted Git invocation.
