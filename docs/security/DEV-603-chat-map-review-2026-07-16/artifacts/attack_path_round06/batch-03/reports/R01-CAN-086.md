## Attack Path Steps

1. Source: Anonymous drug search with a caller-selected 2..100-character name or ingredient.
2. Preconditions: The route is reachable without an external rate control. Attackers generate many distinct legal search strings while live or hybrid upstream calls are configured.
3. Control gap: Six-hour exact-key caching and a 20-result cap reduce repeats but do not limit distinct keys or caller rate.
4. Sink: Each fresh name/ingredient key issues one credentialed government request and creates a Redis entry.
5. Impact: High-cardinality queries can consume provider quota, servlet capacity, and Redis memory.
