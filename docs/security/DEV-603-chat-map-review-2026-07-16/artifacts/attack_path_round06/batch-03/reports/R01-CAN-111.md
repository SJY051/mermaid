## Attack Path Steps

1. Source: Anonymous name search during HYBRID fallback.
2. Preconditions: The live name lookup fails in HYBRID mode. The requested name does not match the fixed fixture product.
3. Control gap: The fallback fixture bypasses the requested item_name filter; caching is exact-keyed but preserves the wrong result for six hours.
4. Sink: Unrelated rows are returned and cached as matches to the attacker/user-selected medicine name.
5. Impact: Users can choose or ground chat on the wrong medicine, and the error persists across callers sharing that query.
