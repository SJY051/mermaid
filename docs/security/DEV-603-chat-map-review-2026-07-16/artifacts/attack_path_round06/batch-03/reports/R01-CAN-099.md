## Attack Path Steps

1. Source: Anonymous pharmacy search in fixture mode or a fixture fallback with arbitrary coordinates.
2. Preconditions: Fixture or fallback data are selected. Caller coordinates differ materially from the fixture capture origin.
3. Control gap: Fixture labeling warns that data are sample/stale, but query parameters are ignored and captured distance values are retained.
4. Sink: FacilityService prefers captured nonnull distances for radius/order and the map describes them relative to the current center.
5. Impact: A remote caller can be told a Seoul fixture pharmacy is only hundreds of metres away, corrupting nearby-care decisions.
