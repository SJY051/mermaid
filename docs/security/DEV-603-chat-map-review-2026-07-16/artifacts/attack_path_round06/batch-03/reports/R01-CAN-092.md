## Attack Path Steps

1. Source: Malformed or compromised HIRA hospital rows carrying a negative finite distance.
2. Preconditions: HIRA or an intermediary supplies a negative distance while coordinates remain parseable. The row reaches the live/hybrid hospital path and is within the returned candidate set.
3. Control gap: Coordinate finiteness is checked, but distance sign/range is not; downstream code trusts any nonnull provider distance.
4. Sink: A negative distance passes radius filtering, sorts first, and is returned/displayed as the caller's distance.
5. Impact: Far or irrelevant hospitals can displace nearby care locations and mislead users choosing where to seek care.
