## Attack Path Steps

1. Source: Unauthenticated GET /api/v1/facilities with type=pharmacy, open_now=true, and attacker-selected coordinates.
2. Preconditions: The route is exposed without an upstream-aware caller budget. Queries select distinct rounded cells and return enough distinct HPIDs to miss both caches.
3. Control gap: Candidate and concurrency caps limit one request to 100 schedule lookups at concurrency four; six-hour caches only help repeated keys.
4. Sink: One cold list lookup plus up to 100 cold weekly-hours calls spend the server's pharmacy credential.
5. Impact: Roughly ten cold populated requests can consume the documented 1,000-call daily budget and deny pharmacy availability to all users.
