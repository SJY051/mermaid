## Attack Path Steps

1. Source: Facility search on a Korean public holiday.
2. Preconditions: The requested current date is a Korean public holiday. A facility's holiday hours differ from its ordinary weekday hours.
3. Control gap: HolidayCalendar is injected as the closest control but always returns false; holiday-specific upstream fields are not used.
4. Sink: WeeklyHours evaluates the ordinary weekday timetable and emits open/closed status.
5. Impact: Users can be sent to a closed facility or steered away from an open one on holidays.
