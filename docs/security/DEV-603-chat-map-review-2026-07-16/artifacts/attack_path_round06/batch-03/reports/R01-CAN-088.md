## Attack Path Steps

1. Source: Anonymous hospital searches with attacker-selected floating-point latitude, longitude, and radius.
2. Preconditions: Production uses the Redis cache manager and the route is externally reachable. No deployment-level Redis maxmemory/eviction or edge admission control compensates for the repository configuration.
3. Control gap: Entries expire after six hours, but exact coordinates form the key and repository cache configuration sets no memory/cardinality bound.
4. Sink: Each distinct tuple stores a hospital batch containing up to 2,000 rows in Redis.
5. Impact: Coordinate churn can fill Redis and repeatedly trigger large HIRA fetches, denying cached care-location data.
