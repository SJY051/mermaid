## Attack Path Steps

1. An anonymous facility request causes the backend to call a fixed government provider.
2. A malicious or compromised provider returns a non-00 envelope with attacker-chosen resultMsg.
3. PublicApiResponse concatenates resultMsg into PublicApiException.
4. Hybrid or live error handling writes that text into operational logs, but user-visible safety output remains generic or fail-closed.
