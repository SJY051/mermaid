## Attack Path Steps

1. An anonymous client posts an allergy under its own already-known deviceId while the profile still has default-off consent.
2. ProfileService constructs an ApiException whose internal message includes that deviceId.
3. ApiException preserves the message and GlobalExceptionHandler writes it to warning logs while redacting the browser response.
4. A distinct log reader that lacks equivalent profile-database authority could obtain and replay the logged capability against the one profile aggregate.
