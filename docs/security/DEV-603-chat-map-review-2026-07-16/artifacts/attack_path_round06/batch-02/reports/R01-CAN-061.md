## Attack Path Steps

1. An anonymous user submits exact latitude and longitude to the facility endpoint.
2. In live mode, a hospital or pharmacy upstream call fails.
3. The client wrapper concatenates the exact coordinates into PublicApiException and GlobalExceptionHandler writes the throwable to operational logs.
4. A distinct log reader without equivalent access to the user's request or location could observe the care-search coordinates.
