## Attack Path Steps

1. An anonymous user submits a symptom or medication question to chat.
2. SearchTermExtractor accepts bounded product names from the user's text and model-derived ingredient terms from that health context.
3. DrugContextRetriever writes those exact lists at INFO, including proposals later suppressed because an allergy was declared.
4. The configured package log level enables persistence outside per-tab sessionStorage; a distinct non-equivalent log reader could inspect the consultation-derived terms.
