## Attack Path Steps

1. An anonymous caller requests /api/v1/drugs/{itemSeq} with an encoded control character in the unconstrained itemSeq.
2. The HTTP stack plausibly decodes the value and DrugService reaches a no-result branch.
3. DrugService concatenates itemSeq into NotFoundException and the global handler logs the message unchanged.
4. If ingress and appender behavior preserve the separator, the caller forges physical warning-log records.
