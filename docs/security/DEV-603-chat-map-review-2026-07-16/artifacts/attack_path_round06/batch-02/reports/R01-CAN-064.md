## Attack Path Steps

1. An unauthenticated caller sends an over-limit profile allergy, alias, memo, or country value containing decoded line breaks.
2. Bean Validation rejects the field while retaining the caller value as rejectedValue.
3. GlobalExceptionHandler logs the validation exception message before returning a sanitized response.
4. If the appender preserves line breaks, the caller persists sensitive text and can create forged physical records for downstream log consumers.
