## Attack Path Steps

1. Anonymous callers issue diverse uncached geocode, facility, or drug requests.
2. A fixed upstream accepts a connection but stalls or trickles an incomplete response.
3. The shared WebClient has no explicit response/read deadline, and callers block or wait through Parallel without a whole-operation cancellation bound.
4. Concurrent stalled operations retain connections, servlet requests, and boundedElastic workers until care-information availability degrades; the service-wide threshold remains unproven.
