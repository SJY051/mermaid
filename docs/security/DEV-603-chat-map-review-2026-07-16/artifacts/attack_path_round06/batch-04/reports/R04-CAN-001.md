## Attack Path Steps

1. Actor/source: A nested .env or .env.* file that is force-staged or already tracked.
2. Control failure: Pre-commit filename matching covers only root .env names and relies on incomplete content heuristics otherwise.
3. Boundary/sink: A nested environment file whose content evades regexes can enter a public commit.
4. Consequence: A live credential can be committed to the public repository.
