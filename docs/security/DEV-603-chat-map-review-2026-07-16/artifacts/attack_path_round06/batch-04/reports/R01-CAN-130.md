## Attack Path Steps

1. Actor/source: A network peer able to reach a developer host running the default Compose MariaDB service.
2. Control failure: Short-form port publishing exposes 3306 beyond loopback while setup copies predictable credentials without forcing replacement.
3. Boundary/sink: The persistent application MariaDB accepts SQL using public fallback credentials.
4. Consequence: A reachable peer can read/alter allergy profiles and saved locations or destroy service availability.
