## Attack Path Steps

1. Public ingress: backend/src/main/java/com/mermaid/facility/GeocodeController.java:22-28 — The public geocode route accepts a caller-selected query and returns GeocodeClient results.
2. Actor/source: Malformed/compromised Naver geocode JSON containing non-finite or out-of-range coordinates.
3. Control failure: Double.parseDouble is followed by no finite or geographic range validation.
4. Boundary/sink: backend/src/main/java/com/mermaid/facility/GeocodeController.java:22-28 — The same controller serializes unchecked non-finite or out-of-range coordinates as a successful response.
5. Consequence: Invalid coordinates can break or misplace map and nearby-care guidance while the API reports success.
