## Attack Path Steps

1. Actor/source: A network peer able to reach default Compose Redis on port 6379.
2. Control failure: Redis runs without authentication/ACL and is published beyond loopback; cached values lack writer integrity.
3. Boundary/sink: Unauthenticated commands can read/delete/replace caches consumed as government-derived drug and facility facts.
4. Consequence: A reachable peer can flush caches, exhaust resources, cause denial of service, and potentially inject serializer-compatible false facts.
