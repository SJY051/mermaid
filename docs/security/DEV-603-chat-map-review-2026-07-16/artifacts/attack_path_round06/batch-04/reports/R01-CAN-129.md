## Attack Path Steps

1. Actor/source: A developer running the local ERD helper after overriding the MariaDB root password in .env.
2. Control failure: The helper passes the root password as a literal docker client argument instead of stdin or a protected option source.
3. Boundary/sink: A permitted same-host process observer can inspect the short-lived docker client argv.
4. Consequence: The local MariaDB root credential may be disclosed to another local principal.
