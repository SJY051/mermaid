## Attack Path Steps

1. Source: A compromised/malformed government facility record supplies markup-like nameKo content; ordinary users do not control this field directly.
2. Control gap: escapeHtml protects the explicit icon HTML string, and React safely renders the list/detail copy; the raw SDK title option is left to third-party implementation behavior.
3. Sink: Naver Marker receives raw untrusted title text; if the SDK interpolates it into HTML rather than using a safe DOM property, scriptable markup can execute in the app origin.
4. Impact: A malicious facility name could execute same-origin JavaScript and steal consultation/allergy/location storage or falsify medical-access UI.
