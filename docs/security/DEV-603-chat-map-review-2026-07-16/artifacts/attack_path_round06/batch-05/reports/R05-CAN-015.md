- Ledger row ID: central-validation-round-06:R05-CAN-015

## Attack Path Steps

1. Source: A valid model answer classifying a non-rule-matched condition as urgent and placing care timing in urgency.title/message.
2. Control gap: The backend preserves a typed urgent state, but both first-party render paths gate the urgency copy on the emergency enum value only.
3. Sink: Urgent title/message never reaches the screen unless the model duplicates it in summary, which no invariant requires.
4. Impact: A user may miss a recommendation to seek prompt care because the structured urgency information is silently discarded.
