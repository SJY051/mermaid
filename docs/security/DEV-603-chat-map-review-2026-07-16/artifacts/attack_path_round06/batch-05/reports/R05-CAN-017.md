## Attack Path Steps

1. Source: A prompt-injection payload or unfaithful provider response on a phrase not caught by deterministic triage.
2. Control gap: The server accepts schema-valid ROUTINE or URGENT classifications without a fail-closed semantic check that the non-emergency classification is true.
3. Sink: A false non-emergency urgency value is copied into the accepted answer and the emergency branch is not activated.
4. Impact: A missed emergency can be presented as routine or urgent, delaying emergency care.
