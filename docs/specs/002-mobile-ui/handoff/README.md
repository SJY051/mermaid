## 구현하며 고칠 것 (검수에서 나온 개선점, 2026-07-13)

- **약국 아이콘**: 현재 선 위주 알약 → 흔히 떠올리는 반반 캡슐로.
- **ER 아이콘**: 번개는 "응급" 전달이 모호 → heart-pulse 계열 검토. 온보딩에서 픽토그램 뜻 1회 설명.
- **보편 아이콘**(전화 등)은 Lucide 등 검증된 세트 사용 — 자체 제작 불필요.
- 아이콘 **선 굵기·라운딩 혼재** → 통일.
- 1d FAR-ZOOM FALLBACK의 약국 open·병원 unknown 아이콘이 **가로로 찌그러짐** → 비율 교정.
- 1t Waiting Stripe 왼쪽 진한 바 애니메이션이 **끊김** → 부드럽게.
- 다크 프레임(1p–1s)은 **theme-neutral에 다크 토큰이 없어 대기** (spec §5).
